# format_with_black_gh_action
use the https://pypi.org/project/black/ with GitHub actions to auto format your code on a push or pull requests
<br>
put https://github.com/mendelsshop/format_with_black_gh_action/blob/main/black.yml into you repo's /.github/workflows/ folder
<br>
if you don'y have this folder you can just create it
